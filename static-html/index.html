Remote Crypto Services

Secure. Fast. Free.

This server hosts freely accessible cryptographic web services.

Protect
Passwords

This server hosts a verifiable, oblivious cryptographic protocol called Pythia at https://remote-crypto.io/pythia/eval . The Pythia protocol protects passwords using oblivious cryptography. Requests are strictly rate-limited to defeat online guessing attacks.

Application Independent

The Pythia protocol is application agnostic. It supports CloudLock, a command-line file encryption application; and SafeID, secure server-side password authentication library.

These services can also be used to secure your password-protected, internet-connected application.

Open Source
& Free

Pythia is open source and free. You can use this development server by downloading an existing application, library, or issuing HTTPS requests from your own application and parsing the JSON responses.

You can also run your own Pythia server using our open source code base.
